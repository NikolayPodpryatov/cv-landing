Веб-приложение для DataMatrix и валидации в ЧЗ

Контекст: при работе с маркетплейсами у склада есть задача печатать дубликаты DataMatrix (маркировка Честный Знак) и клеить на упаковку. Сложность в том, чтобы корректно собрать дубль марки так, чтобы она точно проходила валидацию в ЧЗ. Рынок предлагает единичные решения только в рамках дорогих и комплексных товаро-учетных систем.
Действия:
  • Продумал максимально простой вариант: отсканили оригинальную марку и на выходе получили точную копию, готовую для печати.
  • Изучил документацию по DataMarix именно в ЧЗ (это больно).
  • С помощью ChatGPT написал приложение — валидация не проходила.
  • Разобрался с декодированием символов и как собрать побайтово точную копию, с корректными криптохвостами и всеми нюансами.
  • Добил с помощью агента Copilot до рабочего состояния и успешной проверки.
Результат: сэкономили на дороших решениях и получили лаконичный рабочий вариант: отсканили марку → сгенерировали PDF под печать → прошли проверку на маркетплейсах и в ЧЗ.

Конвертер описаний для контент-менеджера

Контекст: нужно было привести описания карточек товаров на сайте в единый брендовый вид, при этом сохранив гибкость шаблонного блока нашей CMS (текстовый редактор + HTML-редактор в бэк-офисе). Контент-менеджеры не используют HTML и ошибаются в структуре. Нужен был конвертер, который позволит собрать обычный текст менеджера в оформленный HTML для быстрой вставки в админке.
Действия:
  • Продумал структуру описания товара, учитывая специфику продукции.
  • Собрал универсальный HTML / CSS с лаконичными инлайн-стилями.
  • С помощью LLM написал приложение для конвертации текста в уже оформленный HTML.
  • Сохранил гибкость CMS-редактора – легко правим, добавляем уникальные фишки для товара, если необходимо.
Результат: контент-менеджер не ошибается со структурой и все карточки выглядят в единой стилистике. Менеджер заполняет пошагово необходимые секции → получает готовый HTML → при необходимости правит уже в редакторе CMS, даже не разбираясь в разметке.
